# To-do CRUD App

## Overview

This is a simple CRUD (Create, Read, Update, Delete) web app I built while learning backend development.

It lets users:
- Create posts
- View posts
- Edit posts
- Delete posts
- Register and log into an account

The main goal was to understand how frontend, backend, and database work together in a real project instead of just following tutorials.


## Tech Stack

- Backend: PHP  
- Database: Supabase (PostgreSQL)  
- Frontend: HTML, CSS, JavaScript  
- Local Server: XAMPP (Apache)

## What I Practiced

- Basic CRUD operations
- Connecting PHP to a database
- User authentication flow (signup / login)
- Structuring a small full-stack project
- Working with a real database (Supabase)

---

## Notes

This is a learning project, not something production-ready. I mainly built it to get comfortable with how data flows through a full web application.

---

## Running It

To run locally:

1. Put the project inside your XAMPP `htdocs` folder  
2. Start Apache in XAMPP  
3. Set up your Supabase database  
4. Update your DB credentials in the backend PHP files  
5. Open it in the browser via `localhost`
